# Mistral AI Agent Workshop — NYC 2026

Starter repo for the Mistral AI agent workshop.

```
activity2/           Write a skill — teach the agent your standards

templates/           Starter templates for Activity 3
```

## Setup

1. **Install Vibe** — either:
   - **VSCode** (recommended): Extensions > search "Mistral Vibe" > Install
   - **CLI**: `curl -LsSf https://mistral.ai/vibe/install.sh | bash`

2. **Get your API key**:
   - Go to [console.mistral.ai](https://console.mistral.ai)
   - Switch to the **workshop org** (top-left dropdown)
   - API Keys > Create new key > copy it

3. **Configure Vibe** with your API key:
   - VSCode: open Vibe panel > paste key when prompted
   - CLI: `vibe --setup`

4. **Clone this repo**:
   ```
   git clone https://github.com/sam-hartman-mistral/agent-workshop-nyc-starter
   cd agent-workshop-nyc-starter
   ```
