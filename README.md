# Mistral AI Agent Workshop — NYC 2026

Welcome! This is the starter repo for the Mistral AI agent workshop.

## Setup

Before you start, make sure you have:
- [x] Vibe installed (`curl -LsSf https://mistral.ai/vibe/install.sh | bash`)
- [x] Your API key from [console.mistral.ai](https://console.mistral.ai) (switch to the workshop org first)
- [x] Vibe configured (`vibe --setup`)

## Activities

| Activity | What you'll build |
|---|---|
| [Activity 1](./activity1/README.md) | Your first skill — a `/pr-reviewer` |
| [Activity 2](./activity2/README.md) | Wire it into a sub-agent |
| [Activity 3](./activity3/README.md) | Build something for your org (open build) |

We'll walk through Activities 1 and 2 together. Activity 3 is yours.
